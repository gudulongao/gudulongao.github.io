遇到的问题

找不到mysql.sock

初始化的时候，发现报错，在/tmp下找不到mysql.sock。看了解决mysql启动找不到mysql.sock中提到的方式，通过mysql.server起服务会重新生成mysql.sock文件。可是我用mysql.server start报错了，没有起起来但是生成了报错日志。我查看报错日志发现，是我原先有一个mysql服务。我把服务杀死，mysql.server就能起来了，也重新生成了mysql.sock。最后成功初始化。
