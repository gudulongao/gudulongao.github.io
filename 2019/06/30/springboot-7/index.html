Spring Data

Spring Data是Spring用于解决数据访问问题的一系列解决方案。包含大量关系型数据库及非关系型数据库的数据访问解决方案。
Spring Data 提供统一的API来进行数据访问操作而屏蔽了不同数据存储技术的差异。Spring提供Spring Data Commons项目无论使用关系型数据库还是非关系型数据库，都能够有Spring的统一标准，该标准提供CRUD(创建，获取，更新和删除)、查询、排序和分页的相关操作。
Spring Data Commons提供了Spring Data Repository抽象，使用Spring Data Repository可以减少数据访问层的代码。Spring Data Repository提供了基础的接口**Repository、CruRepository和PagingAndSortingRepository。

Repository接口

Repository接口可以接受实体T 和主键id。

CrudRepository接口

CrudRepository接口提供CRUD操作。

PagingAndSortingRepository接口

PagingAndSortingRepository接口提供分页和排序操作。

Spring Data JPA

JPA(Java Persistens API)基于O/R映射的标准规范。即只定义标准规范（注解、接口），不提供实现，软件服务商根据标准规范来提供具体的实现，使用者只需要面向标准规范来使用不用与具体的软件服务商打交道。个人感觉通过JPA可以屏蔽不同的数据库的差异。

Spring Data JPA提供了JpaRepository接口。
